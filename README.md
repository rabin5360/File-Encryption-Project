# File Encryption Project
A terminal-based file encryption system built with Python and MySQL.

## Features
- Encrypt any file using a password
- Decrypt encrypted files
- Logs all actions to a MySQL database

## Technologies
- Python 3
- MySQL
- Cryptography library

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Set up MySQL database (see database.py)
3. Run: `python main.py`
